Bingo Master Board helps you host and manage your own Bingo games.

Set the winning Bingo pattern.
Randomly draw a Bingo ball.
Clear the board for a new game.
Make the board fullscreen.

 

Change how the board looks.
Toggle the balls drawn/remaining counter.
Hide the board for a tougher game.
Manually edit the board if you're drawing balls separately.

Use keyboard shortcuts.

Draw balls with the space bar.

Reset the board with the r key.

Other Shortcuts

f - Toggle fullscreen

t - Go to Themes page

w - Go to Winning Pattern page

enter - Go to master board from title

v - Toggle balls drawn/remaining

x - Hide/show board

Add ?masterboard to the end of the URL to quickly load the master board.

Thanks for using Bingo Master Board! I really appreciate it.
